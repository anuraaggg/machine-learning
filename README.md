# Machine Learning Projects

Machine learning projects with regression algorithms and data analysis using Python.

## Notebooks

1. **numpy_pandas_matplotlib_basics.ipynb** - Data manipulation and visualization basics using NumPy, Pandas, and Matplotlib. Includes Titanic dataset exploration.

2. **linear_regression.ipynb** - Linear regression model to predict body fat percentage from fitness attributes (age, calories, sleep hours, workout sessions).

3. **multiple_regression.ipynb** - Multiple linear regression model to predict house prices from property features (area, bedrooms, bathrooms, age, distance from city center).

## Requirements
```
pandas, numpy, matplotlib, scikit-learn, jupyter
```
